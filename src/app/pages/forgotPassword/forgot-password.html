¿Cómo funciona la recuperación?

  • Ingresas el correo asociado a tu cuenta.
  • El sistema valida si existe un usuario con ese correo.
  • Si es válido, se genera un enlace temporal de recuperación.
  • Recibes un correo con instrucciones para crear una nueva contraseña.
